# Spotlight App

A Streamlit app that simulates the impact of spotlight effect, automation, and blameless culture on predicted recovery time.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment

This app is ready for Streamlit Cloud. Make sure the main file is set to `app.py`.

## Dataset

No external dataset is required. The app uses user-selected inputs and a simple regression-style formula to generate predictions.
